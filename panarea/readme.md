# Isola di Panarea

Остров **Panarea** известна своей атмосферой: коктейли на закате, стильная публика, живая музыка и клубная сцена в высокий сезон. Это самый «богемный» остров Эол.

![alt text](map.png)

Остров с высокими скалистыми берегами, которые создают живописные виды и защищённые бухты для швартовки. Швартовка на якоре или буях в основном с западной стороны острова. Есть много хороших точек для дайвинга в группе мелких островов с западной части Panarea: особенно популярны точки у островков Lisca Bianca.


## San Pietro (восточная часть)
Главный посёлок острова и центр жизни. Белые дома, узкие улочки, бутики, бары и рестораны, лучшее место для вечернего аперитива и прогулок.

![alt text](citi.png)

Для стоянки есть буи южнее пирса.

![alt text](boe.png)

Высокий городской причал часто занят паромами, поэтому яхтсменам рекомендуется использовать буи или якорные стоянки. Городской пирс для высадки с тузика. 

![alt text](San-Pietro.png)

![alt text](San-Pietro2.png)

![alt text](San-Pietro3.png)

Есть несколько продуктовых магазинов. Цены высокие. В **Supermercato Panarea Tesoriero** - прямо у пирса `есть алкоголь`.


## Grotta Polomba (восточная часть ниже San Pietro)
это уединённая бухта с кристально чистой водой, идеально подходящая для снорклинга. Удобное место для швартовки на якоре или буях. Высадка на берег с тузика проблематична.

![alt text](Grotta.png)


## Drauto (юго-восточная часть)

Есть буи для швартовки. Инфраструктуры нет, но есть небольшой пляж, где возможна высадка на берег.

![alt text](Drautto.png)
![alt text](Drautto1.png)

## Milazzese Bay (юго-восточная часть)

Якорная швартовка. Пляж. Возможна высадка на берег. Доисторическое поселение.

## Дайвинг

### Cala Junco (южная часть)

**Cala Junco (Panarea)** — одна из самых известных и красивых бухт и, пожалуй, визитная карточка Panarea. Это почти идеально круглая бухта с кристально чистой, спокойной водой, окружённая вулканическими скалами.

С моря **Cala Junco** — отличное место для дневной якорной стоянки и купания: прозрачная вода, снорклинг, SUP и эффектная цветовая градация от светло‑бирюзового до тёмно‑синего. На берег можно высадиться и пройти пешком к доисторическому поселению Capo Milazzese, расположенному прямо над бухтой.

![alt text](junco.png)
![alt text](junco2.png)

### Dattilo
Крутой скалистый островок (faraglione) у берегов Panarea, один из самых выразительных в группе маленьких островов вокруг неё. Он представляет собой почти вертикальную вулканическую скалу с резкими формами, поднимающуюся прямо из моря, и служит знаковым навигационным ориентиром для яхтсменов.
Швартовка на якоре в западной части. Высадки на берег нет.

![alt text](Dattilo.png)
![alt text](Dattilo1.png)

### Lisca Bianca 
Небольшой скалистый островок рядом с **Dattilo**, известный подводными фумаролами и выходами горячих газов прямо в море. Lisca Bianca — популярное место для купания и снорклинга благодаря подводным фумаролам, создающим эффект бурлящей воды. Якорная стоянка возможна только при спокойном море, обычно для дневного визита, без высадки на берег.
