# ⛵ Эолийские острова — путеводитель для яхтсменов

Путеводитель по Эолийским (Липарским) островам для яхтсменов — марины, якорные стоянки, достопримечательности, рестораны, регатные маршруты и ветровой режим.

🌐 **Сайт:** [true-while.github.io/lipari](https://true-while.github.io/lipari/)

---

## Сицилия

| Порт | Описание |
|------|----------|
| [Milazzo](https://true-while.github.io/lipari/milazzo/) | Главный порт на материке, стартовая точка чартеров |
| [Portorosa](https://true-while.github.io/lipari/portorosa/) | Крупнейшая марина Сицилии, «ворота» к архипелагу |
| [Capo d'Orlando](https://true-while.github.io/lipari/capo-d-orlando/) | Альтернативный старт, низкие цены, парк Неброди |

## Технологии

- [Jekyll](https://jekyllrb.com/) + [Minimal Mistakes](https://mmistakes.github.io/minimal-mistakes/) (тема Air)
- GitHub Pages
- Azure Application Insights (аналитика)

## Локальный запуск

```bash
bundle install
bundle exec jekyll serve
```

Сайт будет доступен по адресу `http://127.0.0.1:4000/lipari/`

## Утилиты

- `resize_images.py` — скрипт для приведения всех изображений (кроме `title.*` и `map.*`) к ширине 1280px
